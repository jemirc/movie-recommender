# Movie Recommender

영화, 사용자, 평점 데이터를 CSV로 관리하고 사용자 간 평점 유사도를 이용해 영화를 추천하는 C++ 콘솔 프로그램입니다. CSE2150 C++ Programming M4 발표를 위해 M3 추천 시스템 위에 장르 필터 추천과 통계 기능을 확장했습니다.

## 빌드 방법

```bash
make
```

빌드 결과로 `movie_recommender` 실행 파일이 생성됩니다.

## 실행 방법

```bash
make run
```

또는 직접 실행할 수 있습니다.

```bash
./movie_recommender
```

프로그램은 시작할 때 `data/movie.csv`, `data/user.csv`, `data/rating.csv`를 읽고, 종료할 때 현재 데이터를 다시 저장합니다.

## 주요 기능

- 영화 추가, 제목 검색, 전체 목록 출력, 평점순 정렬 출력
- 사용자 추가, 사용자 목록 출력
- 사용자별 영화 평점 입력 및 영화별 평점 조회
- 사용자 간 유사도 기반 영화 추천
- 장르 필터가 적용된 영화 추천
- 전체 평균 평점, 장르별 통계, 평점 Top N 영화 출력

## 클래스 구조

- `Movie`: 영화 ID, 제목, 장르, 개봉 연도, 평균 평점과 평점 수를 관리합니다.
- `User`: 사용자 ID, 이름, 이메일 정보를 관리합니다.
- `Rating`: 사용자 ID, 영화 ID, 평점 점수를 관리합니다.
- `BaseManager`: CSV load/save와 size를 위한 공통 인터페이스입니다.
- `MovieManager`: 영화 목록, 검색, 장르 필터, 평점 재계산, 통계 계산을 담당합니다.
- `UserManager`: 사용자 목록, 추가, 검색, CSV 입출력을 담당합니다.
- `RatingManager`: 평점 추가, 수정, 조회, CSV 입출력을 담당합니다.
- `SimilarityCalculator`: 두 사용자의 공통 영화 평점 차이를 기반으로 유사도를 계산합니다.
- `Recommender`: 유사 사용자의 높은 평점 영화를 추천 후보로 만들고 정렬합니다.
- `DisplayManager`: 콘솔 메뉴별 입력과 출력 흐름을 담당합니다.

## 확장 기능

### 장르 필터 추천

메인 메뉴 `10. 장르 필터 추천`에서 사용자 ID와 장르를 입력하면 해당 장르에 속한 추천 영화만 출력합니다. 기존 `9. 사용자별 영화 추천`은 그대로 유지했고, `Recommender::recommend`에 선택 장르 인자를 추가해 기존 호출과 호환되도록 했습니다.

시연 예시:

```text
10
1
액션
```

### 통계 기능

메인 메뉴 `11. 통계 보기`에서 다음 통계를 확인할 수 있습니다.

- 전체 평균 평점과 전체 평점 수
- 장르별 영화 수, 평점 수, 평균 평점, 인기 장르
- 원하는 개수만큼의 평점 Top N 영화

시연 예시:

```text
11
1
2
3
5
0
```

## M4 발표 시연 흐름

1. `make run`으로 프로그램을 실행합니다.
2. `9. 사용자별 영화 추천`에서 기존 추천 기능이 정상 동작하는지 보여줍니다.
3. `10. 장르 필터 추천`에서 사용자 ID `1`, 장르 `액션`을 입력해 확장 기능을 보여줍니다.
4. `11. 통계 보기`에서 전체 평균, 장르별 통계, Top 5 영화를 차례대로 출력합니다.
5. `0. 종료`로 CSV 저장까지 정상 종료되는지 확인합니다.

## 코드 품질 포인트

- 기능 추가는 `feature/genre-filter`, `feature/statistics`, `feature/readme-m4` 브랜치에서 진행했습니다.
- `MovieManager`는 데이터 계산을, `DisplayManager`는 메뉴 입출력을 맡도록 책임을 분리했습니다.
- `std::copy_if`, `std::accumulate`, `std::sort`, `std::map`, 구조적 바인딩을 사용했습니다.
- CSV 파일 열기 실패와 빈 평점 통계 같은 오류 상황은 예외 처리 또는 안내 메시지로 처리합니다.
- `std::unique_ptr<Movie>`를 사용해 영화 객체 주소 안정성과 메모리 관리를 유지합니다.
